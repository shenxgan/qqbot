# 介绍

## 1. 为何需要
主要是因为以下存在的限制：
1. qq 机器人为自己使用的常用账号，而不是小号
    - 优点：加群只需要 qq 机器人加入即可
2. 同一个 qq 号可以手机和 pc 端同时在线，但是不能多个 pc 端在线
    - qq 机器人登录在 linux 端，属于 pc 端
    - 也就是说 qq 机器人登录后无法在电脑上再次登录同账号

## 2. 实现原理
1. 机器人可以获取所有加入群的聊天信息
2. 存储群聊信息，供后端 api 使用
3. 编写前端进行群聊信息展示

## 3. 代码
1. 前端代码：`ws/index.html`，使用 vue
2. 后端代码：`ws/webqq.py`，与机器人的 `websockets` 服务共用后端
